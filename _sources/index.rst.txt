About
-----

ANTIPASTI (ANTIbody Predictor of Affinity from STructural Information) is a Deep Learning model that predicts the binding affinity of antibodies from their three-dimensional structure. The `repository is on GitHub <https://github.com/kevinmicha/ANTIPASTI>`_.

You can follow the `example notebooks <https://github.com/kevinmicha/ANTIPASTI/tree/main/notebooks>`_ to learn how to use the core functionality of ANTIPASTI

If you have already read our :doc:`paper<citing>`, you might want to proceed with the :doc:`installation` or the :doc:`Tutorial <tutorial/index>`.

.. warning:: Please read the ``pathological`` list in any notebook to see which complexes have been excluded (more details can be found in the ANTIPASTI paper).
