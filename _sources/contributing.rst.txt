Contributing
============

You can contribute to ANTIPASTI in many ways and every submission will be greatly appreciated.

Reporting issues or problems
----------------------------

Please use `GitHub issues <https://github.com/kevinmicha/ANTIPASTI/issues>`_ to report bugs. To do so, please include:

- The name and version of your operating system.

- Detailed steps to reproduce the bug.

Implementing new features
-------------------------

Please use `pull requests <https://github.com/kevinmicha/ANTIPASTI/pulls>`_ if you want to modify the master branch.

- Pull requests should include `automated tests <https://github.com/kevinmicha/ANTIPASTI/tree/main/tests>`_.

- The added functions should include their corresponding docstrings.
